B. OVERVIEW 
A simple terminal-based Tic-Tac-Toe game written in C. Two players (X and O) take turns placing their marks on a 3×3 board. The game enforces valid moves, checks for win or draw conditions, displays the board after each turn, and announces the final result.

C. FEATURES
- Two-player mode (X vs O)
- Input validation to prevent invalid moves
- Automatic win detection (rows, columns, diagonals)
- Draw detection when the board is full
- Clear board display after each move
- Lightweight and runs entirely in the terminal

D. TEACH STACK
- Language: C
- Environment: Terminal / Command Line

E. RULES
- Players take turns entering a position (1–9)
- Player X always starts first
- The game ends when a player wins or the board is full (draw)
